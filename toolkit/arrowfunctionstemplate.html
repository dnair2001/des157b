Dog Age Calculator

This function takes in a human age as a parameter and then multiplies this value by 7 in order to calculate the dog age. I created this function by myself because I have a dog and I calculate my dog's actual dog age for fun.

            function dogAge(age){
                console.log(age * 7);
            }

Function Expression

            const dogAge = function(age){
                console.log(age * 7);;
            }

Arrow Function

            const dogAge = age => age * 7;
            console.log(age * 7);
